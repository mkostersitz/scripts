╔══════════════════════════════════════════════════════════════════════════╗
║                        TROUBLESHOOTING                                   ║
╚══════════════════════════════════════════════════════════════════════════╝

PROBLEM: Commands not found
SOLUTION: source ~/.zshrc (or ~/.bashrc)

PROBLEM: Homebrew not found
SOLUTION: Add to PATH (see README.md)

PROBLEM: Icons not showing
SOLUTION: Set terminal font to "FiraCode Nerd Font"

PROBLEM: Starship not loading
SOLUTION: Check if init line in config, re-run setup

PROBLEM: Need to revert
SOLUTION: cp ~/.zshrc.backup.* ~/.zshrc

FOR MORE HELP:
  • Run ./verify.sh for diagnostics
  • Check README.md troubleshooting section
  • Review PLATFORM_GUIDES.md for OS-specific fixes
